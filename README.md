# Resume Builder

**Description:**  
Interactive Resume Builder web app built with HTML, CSS & JS. Enter personal info, skills, education, experience, upload a profile photo, preview your resume in a stylish popup, and export it as a PDF. Fully responsive, modern UI, no backend required.

## Features
- Live resume preview
- Upload profile photo
- Add multiple skills, education & experience
- Stylish, responsive design
- Download resume as PDF
- Fully client-side (no backend)

## How to Use
1. Open `index.html` in your browser.
2. Fill in the form with your details.
3. Click **Generate Resume** to preview.
4. Use **Close / Edit / Download** buttons in the popup.
5. Download your resume as PDF.
